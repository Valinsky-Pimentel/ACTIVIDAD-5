ETIQUETAS DE HTML

¿Qué es HTML?

El Lenguaje de Marcado de Hipertexto (HTML) es el código que se utiliza para estructurar y desplegar una página web y sus contenidos. HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido.

HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera.

¿Qué son las Etiquetas de HTML?

Las etiquetas HTML son palabras clave que definen cómo el navegador web mostrará el contenido. Las etiquetas HTML contienen tres partes principales:

  • Etiqueta de apertura
  • Contenido
  • Etiqueta de cierre
Pero algunas etiquetas HTML no son etiquetas de cierre. Cuando el navegador web lee un documento HTML, lo lee de arriba a abajo y de izquierda a derecha. Todas las etiquetas HTML deben estar encerradas entre < > estos paréntesis. Cada etiqueta HTML tiene propiedades diferentes, por lo cualrealiza diferentes tareas.

Etiquetas de HTML

< dl > Lista de definición, donde las entradas están como en un diccionario o una enciclopedia.
< dt > Define un nombre en una lista de definición
< dd > Define una descripción o valor de un término en una lista de definición
< blockquote > Especifica una cita larga. Sólo debe contener elementos a nivel de bloque, y no sólo texto plano.
< q > Especifica una cita corta.
< cite > Especifica una cita. Puede definirse como título de una obra.
< abbr > Indica una forma abreviada
< address > Especifica un elemento de dirección. La dirección suele aparecer en cursiva.
< sub >Utilizado para definir el texto de los subíndices
< sup > Utilizado para definir el texto en superíndice.
< code >specifica el texto del código de la informático.
< pre > Utilizados para indicar el texto preformateado.

El texto preformateado suele mostrarse en una fuente de tono fijo, con espacios en blanco intactos y sin ajuste de palabras.

< var > Da formato al texto en un documento. Puede incluir una variable en una expresión matemática.
< kbd > Define la entrada del teclado. Es una etiqueta de frase.
< samp > Indica una muestra de la salida de un programa, y el script
< time > Utilizado para mostrar la fecha y la hora
